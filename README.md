# GRA

Это простое приложение для тех, кто хочет вести личный дневник впечатлений от игр, книг, фильмов и другого контента. Я сделал его, чтобы структурировать мысли там, где обычных списков «пройдено/прочитано» уже не хватает.

## Суть проекта

Вместо сухой оценки от 1 до 10, GRA предлагает разложить впечатления по деталям. Вы сами решаете, по каким критериям оценивать контент: для RPG это могут быть лор и квесты, а для музыки — текст и общая атмосфера.

Приложение полностью автономно. Ваши данные не улетают на серверы и не зависят от сторонних сервисов. Все хранится локально на вашем устройстве в базе данных Hive.

## Основные возможности

- Поддержка разных типов контента: игры, аниме, манга, книги, фильмы, музыка и видео.
- Настраиваемые шаблоны: свой список критериев оценки для каждой категории.
- Автоматический бэкап: сохранение копии базы в выбранную папку (раз в день, неделю или месяц).
- Экспорт и импорт: базу можно выгрузить в JSON и перенести на другое устройство.
- Организация: поиск, фильтрация по типам и быстрые вердикты (нравится / нейтрально / нет).

## Установка и запуск

Самый простой способ — скачать готовый файл в разделе **Releases**:
- **Android**: установите `.apk` файл.
- **Linux**: используйте `.AppImage`.

Если вы хотите собрать приложение из исходников, вам понадобится Flutter SDK:
1. Склонируйте репозиторий.
2. Выполните `flutter pub get`.
3. Запустите через `flutter run`.

## Техническая часть

Приложение написано на Flutter. Для хранения данных используется Hive — легкая NoSQL база данных. Также используются пакеты `share_plus` для экспорта, `file_picker` для работы с файлами и `path_provider` для доступа к системным путям.
